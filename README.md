# Ice Cream Revenue Prediction 🍦

This project predicts ice cream shop revenue based on temperature using a machine learning model.

## Project Overview
The goal of this project is to analyze the relationship between temperature and ice cream sales and build a prediction model using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
The dataset contains:
- Temperature
- Revenue

## Steps in the Project
1. Import libraries
2. Load dataset
3. Data exploration
4. Data visualization
5. Train Linear Regression model
6. Evaluate model performance
7. Predict revenue
